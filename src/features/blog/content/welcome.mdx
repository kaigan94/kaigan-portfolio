## Tech Stack

- Next.js 15
- Tailwind CSS v4
- shadcn/ui

## Featured

- Clean & modern design
- Light & Dark mode
- SEO optimization: [JSON-LD schema](https://json-ld.org), sitemap, robots
- AI-friendly [/llms.txt](https://llmstxt.org)
- Spam-protected email
- Installable PWA

## Blog

- MDX & Markdown support
- Access markdown by adding `.md` to URLs, e.g. `/blog/welcome.md`
- Syntax highlighting for better readability
- RSS feed for content distribution
- Dynamic OG images for rich previews

## Component Registry

You can build and reuse your own components, hooks, and pages using the [shadcn CLI](https://ui.shadcn.com/docs/cli).

Each component can be documented through the Blog, featuring:

- **Live Preview**: Interactive demos
- **Code Snippets**: Copy-ready examples
- **Styled Code Blocks**: Dark mode friendly
- **Command Blocks**: One-click copy for install commands (`pnpm`, `npm`, `yarn`, `bun`)
